# wledpixelart
A simple online tool to create pixelart for a WLED 2D 16x16 Matrix and send it to a WLED controller. 

## Used Hardware 

* WS2812B RGB Flexibele 16X16 Pixel Panel Matrix
* Wemos 1D mini flashed with WLED.
* 5V power
